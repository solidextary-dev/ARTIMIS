# ARTIMIS Tauri + React Scaffold

This repository contains a starter scaffold for a desktop app using:

- **Tauri** (Rust shell for desktop packaging)
- **React** (frontend UI)
- **Backend API layer** (Bungie API integration placeholders)
- **SQLite setup** (schema + data access placeholders)
- **Loadouts and Dungeon Stats service folders** (logic placeholders)

## Structure

- `src/`: React frontend source files
- `backend/`: Node/TypeScript-style backend logic placeholders
- `src-tauri/`: Native Tauri/Rust side scaffold

This scaffold intentionally contains placeholders only and does not implement product features yet.
